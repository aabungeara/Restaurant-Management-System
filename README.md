# 🍽️ Restaurant Management System (JavaFX)

## 📌 Overview

This project is a Restaurant Management System developed using JavaFX.
It allows staff to manage tables, menu items, and orders efficiently.

---

## 🚀 Features

* User Login & Sign Up (with validation and MD5 hashing)
* Manage Tables (Add, Search, Sort by capacity)
* Manage Menu (Add, Edit, Delete, Filter, Sort by price)
* Manage Orders (Filter by status, Search, Sort)
* Dynamic updates using Java Streams

---

## 🛠️ Technologies

* Java
* JavaFX
* Streams & Lambda
* File handling (TXT)

---

## 📂 Structure
